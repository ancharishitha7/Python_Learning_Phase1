# A Simple Calculator - Python

This is a basic **Python calculator** that can perform:

1. Addition
2. Subtraction
3. Multiplication
4. Division (integer division)
5. Modulo Division
6. Exponential

It takes user input for the operation and the two numbers, then displays the result.

---

## Features

- Simple command-line interface
- Handles basic arithmetic operations
- Prompts user for input dynamically
- Prints the result of the operation
